Key Behaviors

  • Search: ⌘F focuses the search input. Types instantly to filter by title/subtitle. "Full" checkbox adds chapter content + TOC to search scope.
  • Filter button: Opens a popover with sections for Status, Tags, Rating, Series, Date range. Each section is a multi-select. Active count shown as badge.
  • Sort dropdown: Options: Date created, Title (A-Z), Rating, Progress, Recently read. Current sort label shown inline.
  • View toggle: Grid (current) or List (compact rows with metadata columns).
  • Filter chips: Only visible when filters are active. Each has ✕ to remove. "Clear all" link at end.
  • Keyboard: ⌘F = search, Esc = clear search/filters, Tab = move between controls.
