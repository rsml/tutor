Data Model Details

  • tags: string[] — free-form, lowercase, hyphenated. Added/removed via context menu or a new "Edit tags" dialog. Auto-suggested from existing tags across all books.
  • series: string | undefined — name of the series this book belongs to. Auto-suggested from existing series names.
  • seriesOrder: number | undefined — position within the series (1-indexed). Required when series is set.
  • sortOrder: number | undefined — manual position for drag-and-drop reorder. Stored as a floating-point number (insertions use midpoint between neighbors). Only used when sort mode is "Manual".
  • No separate registry — tags and series names are derived by scanning all books' metadata. No tags.yml or series.yml file.
  • Filter popover — click "Filter" button, popover with 5 sections. Click to toggle. Multi-select within each section (AND across sections, OR within tags). Applied immediately, no "Apply" button.
