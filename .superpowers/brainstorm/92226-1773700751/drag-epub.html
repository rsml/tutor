Key Details

  • Drag-and-drop: Only enabled when sort mode is "Manual". Uses sortOrder (float) in meta.yml. Insertions use midpoint between neighbors. Drag handles (⠿) appear on cards. Active card gets elevated shadow + slight scale.
  • EPUB import entry points: (1) "Import" button in toolbar/header, (2) Drag .epub file onto the library grid, (3) Context menu in Electron (File → Import EPUB).
  • EPUB parsing: Server-side using epub2 npm package. Extracts XHTML chapters → converts to Markdown via a lightweight HTML-to-MD transform. Preserves headings, lists, code blocks, images (saved to chapter dir).
  • Imported book flag: imported: true in meta.yml. Imported books skip the generation flow (no AI needed). They're read-only in terms of content regeneration, but all other features (feedback, quiz, chat, rating, tags) still work.
  • Drag onto expanded series: When a series stack is expanded, you can drag books into/out of the series to change membership and order.
